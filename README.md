# News Portal Dashboard

## How to Run the Frontend

1. **Install dependencies** (using pnpm):
   ```sh
   pnpm install
   ```

2. **Run the development server:**
   ```sh
   pnpm dev
   ```
   The app will be available at [http://localhost:3000](http://localhost:3000).

3. **Build for production:**
   ```sh
   pnpm build
   pnpm start
   ```
